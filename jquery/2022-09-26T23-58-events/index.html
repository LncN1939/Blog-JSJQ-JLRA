jQuery Events

Publicado el 26 de septiembre de 2022

por Jezer Mejía

Al igual que en otras categorías o secciones de jQuery, los eventos de esta librería están diseñados para simplificar la codificación al ofrecer todos los eventos del DOM de JavaScript a través de métodos de un objeto de jQuery.

Enlazar funciones

jQuery puede enlazar o establecer un binding entre un objeto, su evento y una función.

click()

Entre estos métodos de jQuery está el método click que enlaza el evento del Click a una función predefinida por el programador. 

$("#target").click(() => {
  console.log("El elemento #target fue clickeado");
});

blur()

También disponemos del método blur que se activa cuando un elemento HTMl pierde su atención o focus. 

$("button").blur(() => {
  console.log("El botón ya no tiene el focus");
});

focus()

El evento se activa cuando el evento focus del DOM es ejecutado, o sea cuando un elemento obtiene la atención o focus del del usuario. 

$("button").focus(() => {
  console.log("El botón tiene la atención :D");
});

contextmenu()

Este método enlaza una función al evento contextmenu del DOM, o sea cuando se hace un CTRL+Click ó un click derecho. 

$("p").contextmenu(() => {
  console.log("Hiciste click derecho en el párrafo");
});

hover()

Este evento se ejecuta cuando el mouse entra y sale del elemento. Puede definirse una función a llamar en ambos casos, o una función para cuando entra y otra para cuando se sale.

Como tal, es una abreviatura a los métodos mouseenter y mouseleave. 

$("button").hover(
  () => {
    console.log("El botón tiene el mouse encima");
  },
  () => {
    console.log("Ya no tiene el mouse...");
  }
);

Y más...

jQuery provee de tantos métodos de Eventos como Eventos del DOM existen actualmente e inclusive por cada elemento con sus respectivos Eventos únicos. Puede visualizar todos los eventos del DOM en MDN Web Docs | Event reference

Conectores

on()

Los métodos anteriores son esencialmente abreviaturas a otros métodos que manejan los eventos, en especial son una abreviatura a el método on. Este método es el encargado de enlazar una función a uno o más eventos de un elemento del DOM. Así, cuando el evento ocurra, la función será ejecutada.

off()

Este método desenlaza o elimina el enlace de un evento y una función.
