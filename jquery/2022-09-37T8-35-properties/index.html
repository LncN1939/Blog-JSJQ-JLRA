jQuery Properties

Publicado el 27 de septiembre de 2022

por Roire Villavicencio

Aqui se muestran las propiedades generales de jQuery

Propiedades

context

La propiedad de contexto contiene el contexto original pasado a jQuery, que podría ser un contexto de nodo DOM o, si no se pasa ningún nodo, el contexto del documento.

jQuery

La propiedad jQuery devuelve una cadena que contiene el número de versión de jQuery.

jQuery.fx.interval()

La propiedad jQuery.fx.interval se usa para cambiar la velocidad de activación de la animación en milisegundos.

jQuery.fx.off

La propiedad jQuery.fx.off se usa para deshabilitar/habilitar globalmente todas las animaciones.

jQuery.support

La propiedad jQuery.support contiene una colección de propiedades que representan diferentes características o errores del navegador.

length()

La propiedad de longitud contiene la cantidad de elementos en el objeto jQuery.
