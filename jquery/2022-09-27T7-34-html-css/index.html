Aqui se enumera todos los métodos utilizados para manipular HTML y CSS.

Métodos de HTML/CSS

addClass()

El método addClass() agrega uno o más nombres de clase a los elementos seleccionados.

after()

El método after() inserta contenido específico después de los elementos seleccionados.

append()

El método append() inserta contenido específico al final de los elementos seleccionados.

appendTo()

El método appendTo() inserta elementos HTML al final de los elementos seleccionados.

attr()

El método attr() establece o devuelve atributos y valores de los elementos seleccionados.

before()

El método before() inserta contenido específico delante (antes) de los elementos seleccionados.

clone()

El método clone() hace una copia de los elementos seleccionados, incluidos los nodos secundarios, el texto y los atributos.

css()

El método css() establece o devuelve una o más propiedades de estilo para los elementos seleccionados.

detach()

El método detach() elimina los elementos seleccionados, incluido todo el texto y los nodos secundarios. Sin embargo, guarda datos y eventos.

empty()

El método empty() elimina todos los nodos secundarios y el contenido de los elementos seleccionados.

hasClass

El método hasClass() comprueba si alguno de los elementos seleccionados tiene un nombre de clase específico.

height()

El método height() establece o devuelve la altura de los elementos seleccionados.

html()

El método html() establece o devuelve el contenido (innerHTML) de los elementos seleccionados.

insertBefore()

El método insertBefore() inserta elementos HTML antes de los elementos seleccionados.

offset()

El toPrecision() método formatea un número a una longitud especificada.

outerHeight()

El método outerHeight() devuelve la altura exterior del PRIMER elemento coincidente.

position()

El método position() devuelve la posición (en relación con su elemento principal) del primer elemento coincidente.

prepend()

El método prepend() inserta contenido específico al principio de los elementos seleccionados.

prop()

El método prop() establece o devuelve propiedades y valores de los elementos seleccionados.

remove()

El método remove() elimina los elementos seleccionados, incluido todo el texto y los nodos secundarios.

replaceAll()

El método replaceAll() reemplaza los elementos seleccionados con nuevos elementos HTML.

replaceWith()

El método replaceWith() reemplaza los elementos seleccionados con contenido nuevo.

scrollLeft/Top()

El método scrollLeft() o scrollTop() establece o devuelve la posición de la barra de desplazamiento horizontal o vertical para los elementos seleccionados.

text()

El método text() establece o devuelve el contenido de texto de los elementos seleccionados.

toggleClass()

El método toggleClass() alterna entre agregar y eliminar uno o más nombres de clase de los elementos seleccionados.

unwrap()

El método unwrap() elimina el elemento principal de los elementos seleccionados.

val()

El método val() devuelve o establece el atributo de valor de los elementos seleccionados.

witdh()

El método width() establece o devuelve el ancho de los elementos seleccionados.

wrap()

El método wrap() envuelve los elementos HTML especificados alrededor de cada elemento seleccionado.

wrapAll()

El método wrapAll() envuelve los elementos HTML especificados alrededor de todos los elementos seleccionados.

wrapInner()

El método wrapInner() envuelve los elementos HTML especificados alrededor del contenido (innerHTML) de cada elemento seleccionado.
