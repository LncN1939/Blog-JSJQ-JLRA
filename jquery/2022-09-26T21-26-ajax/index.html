jQuery AJAX

Publicado el 26 de septiembre de 2022

por Armando Meza

Los métodos AJAX son útiles para intercanbiar información con un servidor y actualizar partes de una página web sin necesidad de recargarla completamente.

Métodos AJAX de jQuery

$.ajax()

Este método nos permite realizar una petición AJAX (HTTP asíncrono). Toma un objeto como parámetro

Parámetros

Name Value/Description
async Un valor booleano que define si la petición será o no asíncrona.
beforeSend(xhr) Una función que correrá antes de que la petición sea enviada.
cache Un valor booleano que indica si el navegador debería guardar la información en cache.
complete(xhr,status) Una función que correrá después de que la petición sea enviada. Independientemente de si tenga éxito.
contentType El tipo de contenido que se enviará al servidor.
context Especifica el valor de "this".
data Especifica los datos que serán enviados al servidor
dataFilter(data,type) Una función usada para manejar la respuesta de datos sin procesar de la petición.
dataType El tipo de datos esperado como respuesta del servidor.
error(xhr,status,error) Una función a correr si la petición falla.
global Un valor booleano que especifica si disparar un evento AJAX global.
ifModified Un valor booleano que especifica cuándo una petición tiene éxito si la respuesta ha cambiado desde la última petición.
jsonp Una cadena que sobrescribe la función callback en una petición jsonp
jsonpCallback Especifica un nombre para la función callback en una petición jsonp
password Especifica una contraseña que será usada en una petición de autenticación de acceso HTTP
processData Valor booleano que especifica si los datos enviados con la petición serán transformados en una cadena de consulta.
scriptCharset Especifica el conjunto de caracteres de la petición.
success(result,status,xhr) Una función que será llamada cuando la petición tenga éxito.
timeout El tiempo de espera local de la petición en milisegundos
traditional Un valor booleano que especifica si se usará el estilo tradicional de serialización de parámetros.
type Especifica el tipo de petición. (GET o POST)
url Especifica la URL a la que será enviada la petición.
username Especifica un nombre de usuario que será usado en una petición de autenticación de acceso HTTP
xhr Una función usada para crear el objeto XMLHttpRequest 
$.ajax({
  url: "https://jsonplaceholder.typicode.com/todos/1",
  success: function (result) {
    console.log(result);
  },
});
// Respuesta
// {
//   "userId": 1,
//   "id": 1,
//   "title": "delectus aut autem",
//   "completed": false
// }

$.ajaxSetup()

Establece parámetros por defecto para todas futuras peticiones AJAX 

$.ajaxSetup({
  url: "https://jsonplaceholder.typicode.com/todos/1",      
  success: function (result) {
    console.log(result);
  },
});
$.ajax();
// Respuesta
// {
//   "userId": 1,
//   "id": 1,
//   "title": "delectus aut autem",
//   "completed": false
// }

$.get()

Carga datos de un servidor usando una petición AJAX HTTP del tipo GET 

$.get("https://jsonplaceholder.typicode.com/todos/2");
// Respuesta
// {
//   "userId": 1,
//   "id": 2,
//   "title": "quis ut nam facilis et officia qui",
//   "completed": false
// }

$.getJSON()

Obtiene datos JSON usando una petición AJAX

$.getJSON("https://jsonplaceholder.typicode.com/todos/1");
// Respuesta
// {
//   "userId": 1,
//   "id": 1,
//   "title": "delectus aut autem",
//   "completed": false
// }

$.getScript()

Obtiene y ejecuta javascript usando una petición AJAX

$.getScript("ejemplo.js");

$.post()

Carga datos de un servidor usando una petición AJAX HTTP del tipo GET 

$.post("https://jsonplaceholder.typicode.com/posts", {
  userId: 1,
  id: 2,
  title: "Lorem",
  completed: true,
});

ajaxComplete()

Especifica una función a ejecutar cuando la petición AJAX se complete. 

$(document).ajaxComplete(function(){...})

ajaxError()

Especifica una función a ejecutar cuando la petición AJAX se complete con error. 

$(document).ajaxError(function(){...})

ajaxSend()

Especifica una función a ejecutar antes de que la petición AJAX se envíe. 

$(document).ajaxSend(function(){...})

ajaxStart()

Especifica una función a ejecutar cuando la primera petición AJAX empieza. 

$(document).ajaxStart(function(){...})

ajaxStop()

Especifica una función a ejecutar cuando todas las peticiones AJAX hayan sido completadas. 

$(document).ajaxStop(function(){...})

ajaxSuccess()

Especifica una función a ejecutar cuando todas las peticiones AJAX se hayan completado con éxito. 

$(document).ajaxSuccess(function(){...})

load()

Carga datos de un servidor y los coloca en los elementos seleccionados 

// <div></div>
$("div").load("https://jsonplaceholder.typicode.com/todos/1");
// Resultado
// <div>{ "userId": 1, "id": 1, "title": "delectus aut autem", "completed": false }</div>

serialize()

Crea una cadena de texto codificada en URL como resultado de serializar valores de un formulario. 

// <form>
//   <input type="text" name="nombre" value="Armando">
//   <input type="text" name="apellido" value="Meza"> 
//   <input type="submit" value="Submit">
// </form>
$("form").serialize();
// Resultado
// "nombre=Armando&apellido=Meza"

serializeArray()

Crea un array de objetos como resultado de la serialización de valores en un formulario. 

// <form>
//   <input type="text" name="nombre" value="Armando">
//   <input type="text" name="apellido" value="Meza">
//   <input type="submit" value="Submit">
// </form>
$("form").serialize();
// Resultado
// [
//   {"name": "nombre", "value": "Armando"},
//   {"name": "apellido", "value": "Meza"}
// ]
