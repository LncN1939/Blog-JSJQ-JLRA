Métodos Misceláneos de jQuery

data()

Agrega u obtiene datos de los elementos seleccionados

$("div.data").data("x", 1);
$("div.data").data("x");
// Resultado
// 1

get()

Obtiene el elemento del DOM definido por el selector

// <p>0</p>
// <p>1</p>
// <p>2</p>
// <p>3</p>
$("p").get(0);
// Resultado
// <p>0</p>

index()

Retorna el índice de posición de un elemento específico en relación a otros 

// <p class="primero"></p>
// <p class="segundo"></p>
// <p class="tercero"></p>
$("p.tercero").index();
// Resultado
// 2

$.noConflict()

Libera el control de jquery sobre la variable '$', puede usarse para especificar otro nombre para la variable jquery 

var jq = $.noConflict();
jq("div").first();

$.param()

Crea una representación serializada de un array o un objeto.

const persona = 
{ nombre: "Armando", apellido: "Meza" };
$.param(persona);
// Resultado
// "nombre=Armando&apellido=Meza"

removeData()

Elimina datos almacenados previamente con data()

$("div.data").data("x", 1);
$("div.data").data("x");
// 1
$("div.data").removeData("x");
$("div.data").data("x");
// undefined

toArray()

Retorna los elementos DOM contenidos en un conjunto de jQuery en forma de array. 

// <div>
//   <h1></h1>
//   <h2></h2>
//   <h3></h3>
// </div>
$("div").children().toArray();
// Resultado
// [<h1></h1>, <h2></h2>, <h3></h3>]
