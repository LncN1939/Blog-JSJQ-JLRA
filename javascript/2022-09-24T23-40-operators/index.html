Publicado el 25 de septiembre del 2022

por Leo Corea

Los operadores son de vital importancia para a la hora de asignar valores y compararlos, su conocimiento es de vital importancia a la hora de realizar operaciones aritmeticas que faciliten el manejo de los mismos.

Operadores aritmeticos

Son aquellos que se emplean para operaciones aritmeticas entre variables y/o valores

Suponiendo que y = 6 se explicaran los valores aritmeticos:

Operador Descripción Ejemplo Resultado en y Resultado en x
+ Suma x = y + 2 y = 6 x = 8
- Resta x = y - 2 y = 6 x = 4
* Multiplicacion x = y * 2 y = 6 x = 12
** Exponenciación x = y ** 2 y = 6 x = 36
/ Division x = y / 2 y = 6 x = 3
% Modulo x = y % 2 y = 6 x = 0
++ Incremento x = ++y
x = y++ 		y = 7
y = 7 		x = 7
x = 6
-- Decremento x = --y
x = y-- 		y = 5
y = 5 		x = 5
x = 6

Operadores bit a bit

Los operadores de bits funcionan en números de 32 bits. Cualquier operando numérico en la operación se convierte en un número de 32 bits. El resultado se vuelve a convertir en un número de JavaScript.

Operador Descripción Ejemplo Igual a Resultado Decimal
& AND x = 5 & 1 0101 & 0001 0001 1
| OR x = 5 | 1 0101 | 0001 0101 5
~ NOT x = ~ 5 ~ 0101 1010 10
^ XOR x = 5 ^ 1 0101 ^ 0001 0100 4
<< Shift izquierdo x = 5 << 1 0101 << 1 1010 10
>> Shift derecho x = 5 >> 1 0101 >> 1 0010 2

Nota

Los ejemplos anteriores usan ejemplos sin firmar de 4 bits. Pero JavaScript usa números con signo de 32 bits. Debido a esto, en JavaScript, ~ 5 no devolverá 10. Devolverá -6, es decir no devolvera:

~00000000000000000000000000000101

Si no que devolvera: 111111111111111111111111111111010

Operador typeof

Devuelve el tipo de una variable, objeto, función o expresión:

typeof = "John"         // Retorna una cadena 

typeof = 3.14           // Retorna un numero 

typeof = "[1,2,3,4]"    //  Retorna un objecto

Operador eliminacion

El operador de eliminación elimina una propiedad de un objeto :

const auto ={
    marca = ferrari,
    maximaVelocidad = 150,
    garantia = 10,
    color = azul
};

delete auto.garantia;   // Elimina la tanto el valor como la propiedad

Operador in

El operador in devuelve verdadero si la propiedad especificada está en el objeto especificado; de lo contrario, es falso:

const auto = {marca:"ferrari", color:"azul", garantia:10};

"marca" in auto         // Returns true 

"color" in auto         // Returns true

La instancia del Operador

El operador instanceof devuelve verdadero si el objeto especificado es una instancia del objeto especificado: 

const cars = ["Saab", "Volvo", "BMW"];


(cars instanceof Array)   // Returns truebr 

(cars instanceof String)  // Returns false

El operador del vacío

El operador void evalúa una expresión y devuelve undefined . Este operador se usa a menudo para obtener el valor primitivo indefinido, usando "void(0)" (útil cuando se evalúa una expresión sin usar el valor devuelto).
