JavaScript String

Publicado el 26 de septiembre de 2022

por Leo Corea

Las cadenas en JavaScript son utiles para guardar datos texuales, Una cadena puede almacenar una serie de caracteres como "John Doe", asi mismo puede ser cualquier texto entre comillas simples o dobles: 

let counName1 = "Nicaragua"
let counName2 = "Germany"

Sus indices estan basados en 0, es decir el primero esta en 0, el segundo en 1 y asi sucesivamente

Propiedades y metodos

Normalmente, cadenas como "John Doe" no pueden tener métodos o propiedades porque no son objetos. Pero con JavaScript, los métodos y las propiedades también están disponibles para cadenas, porque JavaScript trata las cadenas como objetos cuando ejecuta métodos y propiedades.

Metodos de cadena

charAt()

El método charAt() devuelve el carácter en un índice especificado (posición) en una cadena.El índice del primer carácter es 0, el segundo 1

Sintaxis

string.charAt(index)

Parámetros

Parámetro Descripción
index El index (posición) del caracter, por default es 0

Ejemplo

let text = "Hello World";
let letter = text.charAt(0);

Se obtendra el primer caracter de una cadena

charAt()

El método charCodeAt() devuelve el Unicode del carácter en un índice especificado (posición) en una cadena.
El índice del primer carácter es 0, el segundo es 1, ....
El índice del último carácter es la longitud de la cadena: 1 (consulte los ejemplos a continuación).

Sintaxis

string.CodeAt(index)

Parámetros

Parámetro Descripción
index El index (posición) del caracter, por default es 0

Ejemplo

let text = "Hello World";
let uni = text.charCodeAt(0);

Se obtendra el unicode del primer caracter de la una cadena

codePointAt()

El método codePointAt() devuelve el valor Unicode en un índice (posición) en una cadena. El índice de la primera posición es 0, el segundo es 1, ....

Diferencia entre charCodeAt() y codePointAt()

charCodeAt() es UTF-16 y codePointAt() es Unicode. charCodeAt() devuelve un número entre 0 y 65535. Ambos métodos devuelven un número entero que representa el código UTF-16 de un carácter, pero solo codePointAt() pueden devolver el valor completo de un valor Unicode mayor que 0xFFFF (65535).

Sintaxis

string.codePointAt(index)

Parámetros

Parámetro Descripción
index El index (posición) del caracter, por default es 0

Ejemplo

let text = "Hello World";
let code = text.codePointAt(0);

Se obtendra el primer caracter de una cadena

concat()

El metodo concat() une 2 o mas cadenas, las cuales no cambia su sintaxis y regresa una cadena complemente nueva

Sintaxis

string.concat(string1, string2, ..., stringX)

Parámetros

Parámetro Descripción
string1, strng2, ... stringX Las cadenas que se van a unir

Ejemplo

let text1 = "Hola";
let text2 = "leo";
let result = text1.concat(text2);

Se obtendra el primer caracter de una cadena

constructor

La propiedad constructordevuelve la función que creó el prototipo String.
Para las cadenas de JS, el constructir retorna: 

function String() { [código nativo] }

Sintaxis

string.constructor

Ejemplo

let message = "Hello World!";
let text = message.constructor;

endsWith()

El método endsWith() devuelve true si una cadena termina con una cadena especificada. De lo contrario retorna false.
El endsWith() es un método distingue entre mayúsculas y minúsculas.

Sintaxis

string.endsWith(searchvalue, length)

Parámetros

Parámetro Descripción
searchvalue La cadena a buscar
length La longitud de la cadena a buscar, es su valor predeterminado

Ejemplo

let text = "Hello world";
let result = text.endsWith("world");

Compruebe si una cadena termina con "wordl"

fromCharCode()

El método fromCharCode() convierte valores de unicode en caracteres, es un metodo estatico de String cuya sintaxis siempre sera String.fromCharCode()



Nota: No puedes usar myString.fromCharCode().

Sintaxis

String.fromCharCode(n1, n2, ..., nX)

Parámetros

Parámetro Descripción
n1, n2, nX Uno o mas valores de unicode a convertir.

Ejemplo

let char = String.fromCharCode(76);
