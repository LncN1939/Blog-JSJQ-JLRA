JavaScript String

Publicado el 26 de septiembre de 2022

por Leo Corea

Las cadenas en JavaScript son utiles para guardar datos texuales, Una cadena puede almacenar una serie de caracteres como "John Doe", asi mismo puede ser cualquier texto entre comillas simples o dobles: 

let counName1 = "Nicaragua"
let counName2 = "Germany"

Sus indices estan basados en 0, es decir el primero esta en 0, el segundo en 1 y asi sucesivamente

Propiedades y metodos

Normalmente, cadenas como "John Doe" no pueden tener métodos o propiedades porque no son objetos. Pero con JavaScript, los métodos y las propiedades también están disponibles para cadenas, porque JavaScript trata las cadenas como objetos cuando ejecuta métodos y propiedades.

Metodos de cadena

charAt()

El método charAt() devuelve el carácter en un índice especificado (posición) en una cadena.El índice del primer carácter es 0, el segundo 1

Sintaxis

string.charAt(index)

Parámetros

Parámetro Descripción
index El index (posición) del caracter, por default es 0

Ejemplo

let text = "Hello World";
let letter = text.charAt(0);

Se obtendra el primer caracter de una cadena

charAt()

El método charCodeAt() devuelve el Unicode del carácter en un índice especificado (posición) en una cadena.
El índice del primer carácter es 0, el segundo es 1, ....
El índice del último carácter es la longitud de la cadena: 1 (consulte los ejemplos a continuación).

Sintaxis

string.CodeAt(index)

Parámetros

Parámetro Descripción
index El index (posición) del caracter, por default es 0

Ejemplo

let text = "Hello World";
let uni = text.charCodeAt(0);

Se obtendra el unicode del primer caracter de la una cadena

codePointAt()

El método codePointAt() devuelve el valor Unicode en un índice (posición) en una cadena. El índice de la primera posición es 0, el segundo es 1, ....

Diferencia entre charCodeAt() y codePointAt()

charCodeAt() es UTF-16 y codePointAt() es Unicode. charCodeAt() devuelve un número entre 0 y 65535. Ambos métodos devuelven un número entero que representa el código UTF-16 de un carácter, pero solo codePointAt() pueden devolver el valor completo de un valor Unicode mayor que 0xFFFF (65535).

Sintaxis

string.codePointAt(index)

Parámetros

Parámetro Descripción
index El index (posición) del caracter, por default es 0

Ejemplo

let text = "Hello World";
let code = text.codePointAt(0);

Se obtendra el primer caracter de una cadena

concat()

El metodo concat() une 2 o mas cadenas, las cuales no cambia su sintaxis y regresa una cadena complemente nueva

Sintaxis

string.concat(string1, string2, ..., stringX)

Parámetros

Parámetro Descripción
string1, strng2, ... stringX Las cadenas que se van a unir

Ejemplo

let text1 = "Hola";
let text2 = "leo";
let result = text1.concat(text2);

Se obtendra el primer caracter de una cadena

constructor

La propiedad constructordevuelve la función que creó el prototipo String.
Para las cadenas de JS, el constructir retorna: 

function String() { [código nativo] }

Sintaxis

string.constructor

Ejemplo

let message = "Hello World!";
let text = message.constructor;

endsWith()

El método endsWith() devuelve true si una cadena termina con una cadena especificada. De lo contrario retorna false.
El endsWith() es un método distingue entre mayúsculas y minúsculas.

Sintaxis

string.endsWith(searchvalue, length)

Parámetros

Parámetro Descripción
searchvalue La cadena a buscar
length La longitud de la cadena a buscar, es su valor predeterminado

Ejemplo

let text = "Hello world";
let result = text.endsWith("world");

Compruebe si una cadena termina con "wordl"

fromCharCode()

El método fromCharCode() convierte valores de unicode en caracteres, es un metodo estatico de String cuya sintaxis siempre sera String.fromCharCode()



Nota: No puedes usar myString.fromCharCode().

Sintaxis

String.fromCharCode(n1, n2, ..., nX)

Parámetros

Parámetro Descripción
n1, n2, nX Uno o mas valores de unicode a convertir.

Ejemplo

let char = String.fromCharCode(76);

includes()

El método includes() devuelve true si una cadena contiene una cadena especificada.De lo contrario vuelve false. Distingue entre mayúsculas y minúsculas.

let text = "Sandino, defensor de la soberania nacional.";
let result = text.includes("Sandino");

indexOf()

El método indexOf() devuelve la posición de la primera aparición de un valor en una cadena, con un retorno de -1 si no se encuentra el valor asi mismo distingue entre mayúsculas y minúsculas..

let text = "Sandino, defensor de la soberania nacional.";
let result = text.indexOf("Sandino");

lastIndexOf()

El método lastIndexOf() devuelve el índice (posición) de la última aparición de un valor especificado en una cadena, busca la cadena desde el final hasta el principio, devuelve el índice desde el principio (posición 0), asi mismi devuelve -1 si no se encuentra el valor Distinguiendo entre mayúsculas y minúsculas.

let text = "Hola Saladino, como has estado Saladino";
let result = text.lastIndexOf("Saladino");

lastIndexOf()

La propiedad length devuelve la longitud de una cadena, siendo la propiedad vacia de una cadena vacía es 0.

let text = "Somos UCA!";
let length = text.length;

localeCompare()

El método localeCompare() compara dos cadenas en la configuración regional actual,devuelve el orden de clasificación -1, 1 o 0 (para antes, después o igual).

let text1 = "UCA";
let text2 = "UNI";
let result = text1.localeCompare(text2);

localeCompare()

El método match() compara una cadena con una expresión regular **, devuelve una matriz con las coincidencias y devuelve nulo si no se encuentra ninguna coincidencia.

let text = "La tierra es un satélite de la luna";
text.match("luna");

function soldado(name, range, born) {

  this.range = range;
  this.born = born;
}
employee.prototype.salary = 2500;
            
const renov = new employee("Renov L.", "Capitan", 1917);

repeat()

El método repeat() devuelve una cadena con un número de copias de una cadena.devuelve una nueva cadena y no cambia la cadena original.

let text = "Buenas tardes!";
let result = text.repeat(3);

replace()

El método replace() busca en una cadena un valor o una expresión regular, devuelve una nueva cadena con los valores reemplazados, asi mismo el método no cambia la cadena original.

let text = "HTML es un lenguaje de programacion";
let result = text.replace("HTML", "JavaScript");

search()

El método search() compara una cadena con una expresión regular **, devuelve el índice (posición) de la primera coincidencia, asi mismo devuelve -1 si no se encuentra ninguna coincidencia y el metodo distingue entre mayúsculas y minúsculas.

let text = "Mr. Dominguez tiene una asador de carne";
let position = text.search("carne");

slice()

El método slice() extrae una parte de una cadena, devuelve la parte extraída en una nueva cadena, no cambia la cadena original.

Los parámetros de inicio y fin especifican la parte de la cadena que se va a extraer. La primera posición es 0, la segunda es 1, ... Y un número negativo selecciona desde el final de la cadena.

let text = "Pedro PicaPiedra";
let result = text.slice(0, 5);

startsWith()

El método startsWith() devuelve true, Si una cadena comienza con una cadena especificada, de lo contrairo devolvera false, distingue entre minusculas y mayusculas.

let text = "Bienvenido a casa tenga un buen viaje.";
text.startsWith("Bienvenido");

toLocaleLowerCase()

El método aLocaleLowerCase() convierte una cadena en letras minúsculas, utilizando la configuración regional actual. La configuración regional se basa en la configuración de idioma del navegador.El toLocaleLowerCase() método no cambia la cadena original.

Asi mismo toLocaleUpperCase(), realiza la misma funcionalidad con diferencia que realiza la conversion de una caden en letras mayusculas.

toLocaleUpperCase()

let text = "Viva la vida";
let result = text.toLocaleUpperCase();

toLocaleLowerCase()

let text = "Viva la vida";
let result = text.toLocaleLowerCase();

toLocaleLowerCase()

El método aLowerCase() convierte una cadena en letras minúsculas, y no cambia la cadena original.

Asi mismo toUpperCase(), realiza la misma funcionalidad con diferencia que realiza la conversion de una caden en letras mayusculas.

Diferencia con toLocaleLowerCase() & toLocaleUpperCase()

Las configuraciones regionales que entran en conflicto con las asignaciones de mayúsculas y minúsculas regulares de Unicode (como el turco), afectan al toLocaleLowerCase y toLocaleUpperCase

toLocaleUpperCase()

let text = "Viva la vida";
let result = text.toUpperCase();

toLocaleLowerCase()

let text = "Viva la vida";
let result = text.toLowerCase();

toString()

El método toString() devuelve una cadena como una cadena, el método no cambia la cadena original, se puede utilizar para convertir un objeto de cadena en una cadena. 

let text = "Hola mundo";
let result = text.toString();

trim()

El método trim() elimina los espacios en blanco de ambos lados de una. 

let text = "    Hola mundo";
let result = trim();

valueOf()

El método valueOf() devuelve el valor primitivo de una cadena, no cambia la cadena original. Se puede utilizar para convertir un objeto de cadena en una cadena.. 

let text = "Viva Nicaragua";
let result = value();
