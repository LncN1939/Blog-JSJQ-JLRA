Javascript Arrays

Publicado el 25 de septiembre de 2022

por Armando Meza

Los Arrays en javascript son usados para guardar múltiples valores en una sola variable. 

const lenguajes = ["HTML", "CSS", "Javascript", "PHP"];

Métodos y propiedades de los Arrays

concat()

Junta varios arrays y los retorna en uno solo, sin modificar ninguno de los existentes. 

const lenguajesFrontend = ["HTML", "CSS", "Javascript"];
          const lenguajesBackend = ["PHP", "Java", "Ruby"];
          const lenguajes = lenguajesFrontend.concat(lenguajesBackend);
          ["HTML", "CSS", "Javascript", "PHP", "Java", "Ruby"]

copyWithin()

Copia elementos del array dentro de este mismo, desde y a posiciones específicas. 

const lenguajes = ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.copyWithin(2, 0);
          ["HTML", "CSS", "Javascript", "Javascript"];

entries()

Retorna un objeto de llave/valor para ser iterado.

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          const len = lenguajes.entries();
          for (let x of len) {
            console.log(x);
          }
          // HTML, CSS, Javascript, PHP.

every()

Chequea si todos los elementos en un array cumplen cierta condición 

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          lenguajes.every((l) => l.length > 3);      
          // False

fill()

Llena los elementos de un array con un valor estático

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          lenguajes.fill("Ruby", 2, 4);
          // ["HTML", "CSS", "Ruby", "Ruby"]

filter()

Crea un nuevo array con los elementos que cumplen cierta condición 

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP", "Ruby"];
          lenguajes.filter((l) => l.length > 3);     
          // ["HTML", "Javascript", "Ruby"]

find()

Retorna el valor del primer elemento en un array que cumple cierta condición 

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP", "Ruby"];
          lenguajes.find((l) => l.length < 4);       
          // "CSS"

findIndex()

Retorna el índice del primer elemento en un array que cumple cierta condición 

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP", "Ruby"];
          lenguajes.findIndex((l) => l.length < 4);  
          // 1

forEach()

Llama una función por cada elemento de un array.

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          lenguajes.forEach((l) => {
            console.log(`${l} es un lenguaje para programación web`);
          });
          // HTML es un lenguaje para programación web.
          // CSS es un lenguaje para programación web.
          // Javascript es un lenguaje para programación web.
          // PHP es un lenguaje para programación web.

from()

Crea un array a partir de un string.

Array.from("Hola");
          // ["H", "o", "l", "a"]

includes()

Chequea si un elemento existe dentro de un array.

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          lenguajes.includes("C++");
          // False

indexOf()

Busca un elemento en el arreglo y retorna su posición, o -1 si no existe. 

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          lenguajes.includes("C++");
          // False

isArray()

Chequea si un objeto es de tipo array.

Array.isArray(["HTML", "CSS", "Javascript"]);   
          // true
          Array.isArray("HTML, CSS, Javascript");
          // false

join()

Junta todos los elementos de un array en una cadena.

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          lenguajes.join();       
          // "HTML,CSS,Javascript,PHP"

keys()

Retorna un objeto de iteración de array conteniendo las llaves del array original. 

const lenguajes = ["HTML", "CSS", "Javascript", 
          "PHP"];
          keys = lenguajes.keys();
          for (let x of keys) {
            console.log(x);
          }
          // 0
          // 1
          // 2
          // 3

lastIndexOf()

Busca un elemento en el array, empezando por el fin, y retorna su posición. 

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP", "HTML", "CSS"];
          lenguajes.lastIndexOf("HTML");
          // 4

length

Retorna el número de elementos de un array.

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.lastIndexOf("HTML");
          // 4

map()

Crea un nuevo array en base a una función que se llama por cada elemento. 

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.map((l) => l.toLowerCase());
          // ["html", "css", "javascript", "php"]

pop()

Elimina el último elemento de un arreglo y lo retorna.

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.pop(1);
          // "CSS"

push()

Agrega un nuevo elemento al final del arreglo y retorna su nuevo tamaño 

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.push("Ruby");
          // 5

reduce()

Reduce los valores de un array a un solo valor. Recorre el array de izquierda a derecha. 

const numeros = [80, 24, 129];
          numeros.reduce((total, num) => total - num);
          // -73

reduceRight()

Reduce los valores de un array a un solo valor. Recorre el array de derecha a izquierda. 

const numeros = [80, 24, 129];
          numeros.reduceRight((total, num) => total - num);
          // 25

reverse()

Revierte el orden de los valores en un arreglo.

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.reverse();
          // ["PHP", "Javasript", "CSS", "HTML"]

shift()

Elimina el primer elemento de un array, y retorna ese elemento.

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.shift();
          // "HTML"

slice()

Selecciona una parte de un array y retorna un nuevo array.

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.slice(0, 2);
          // ["HTML", "CSS"]

some()

Chequea si algún elemento del array cumple con cierta condición

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.some((l) => l.length > 3);
          // true

sort()

Ordena los elementos en un array.

const lenguajes = 
          ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.sort();
          ["CSS", "HTML", "Javascript", "PHP"];

splice()

Añade o elimina elementos de un array. Modifica el array original y retorna un array con los elementos eliminados. 

const lenguajes = ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.splice(2, 2, "Ruby", 
          "C++");
          // ['Javascript', 'PHP']

toString()

Convierte el arreglo a una cadena y retorna el resultado.

const lenguajes = ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.toString();
          // "HTML,CSS,Javascript,PHP"

unshift()

Añade elementos al inicio de un array, y retorna el tamaño nuevo del array. 

const lenguajes = ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.unshift("Ruby");
          // 5
          // ["Ruby", "HTML", "CSS", "Javascript", "PHP"]

valueOf()

Retorna el valor primitivo de un array.

const lenguajes = ["HTML", "CSS", "Javascript", "PHP"];
          lenguajes.valueOf();
          // 5
          // ["HTML", "CSS", "Javascript", "PHP"]
