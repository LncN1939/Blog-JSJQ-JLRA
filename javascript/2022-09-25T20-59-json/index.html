JSON es un formato para almacenar y transportar datos. JSON es texto, puede ser transportado hacia cualquier lugar, y leído por cualquier elnguanje de programación. Los objetos de javascript pueden ser convertidos a JSON, y viceversa.

Métodos JSON

parse()

Analiza una cadena JSON y retorna un objeto de javascript.

const cadena =        
          '{"nombres":"Armando Alexander","apellidos":"Meza López","edad":21,"altura":1.64,"intereses":["Tocar guitarra","Leer"],"esEstudiante":true}';
        JSON.parse(cadena);

stringify()

Convierte un objeto de javascript a una cadena JSON.

const persona = {
            nombres: "Armando Alexander",
            apellidos: "Meza López",       
            edad: 21,
            altura: 1.64,
            intereses: ["Tocar guitarra", "Leer"],
            esEstudiante: true,
          };
          JSON.stringify(persona);
          // '{"nombres":"Armando Alexander","nombre":"Meza López","edad":21,"altura":1.64,"intereses":["Tocar guitarra","Leer"],"esEstudiante":true}'

Tipos de dato válidos

  • Cadenas
  • Números
  • Objetos
  • Arreglos
  • Booleano
  • null
