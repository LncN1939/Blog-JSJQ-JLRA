JavaScript Statements

Publicado el 27 de septiembre de 2022

por Roire Villavicencio

Las declaraciones de JavaScript suelen comenzar con un statements de declaración para identificar la acción de JavaScript que se va a realizar. Los statements de declaraciones son palabras reservadas y no se pueden usar como nombres de variables (o cualquier otra cosa).

Métodos de Statements

break()

La break declaración puede usar una referencia de etiqueta para salir de cualquier bloque de código JavaScript.

class()

A class es un tipo de plantilla de objeto. La class declaración inicia una clase de JavaScript.

const()

La const instrucción declara una variable. Las variables son contenedores para almacenar información.

continue()

La continue instrucción interrumpe una iteración (en el bucle) si se produce una condición específica y continúa con la siguiente iteración del bucle.

debugger()

La debugger instrucción detiene la ejecución de JavaScript y llama al depurador.

do...while()

El do...while combo de instrucciones define un bloque de código que se ejecutará una vez y se repetirá siempre que una condición sea true.

for()

La for instrucción define un bloque de código que se ejecuta siempre que una condición sea true.

for...in()

El for...in combo de declaraciones itera (bucles) sobre las propiedades de un objeto.

for...of()

El for...of combo de declaraciones itera (bucle) sobre los valores de cualquier iterable.

function()

Una función declarada se "guarda para su uso posterior" y se ejecutará más tarde, cuando se invoque (llame).

if...else()

La declaración if/else ejecuta un bloque de código si una condición especificada es verdadera. Si la condición es falsa, se puede ejecutar otro bloque de código.

let()

La let instrucción declara una variable. Las variables son contenedores para almacenar información.

return()

La return instrucción detiene la ejecución de una función y devuelve un valor.

switch()

La declaracion switch es una parte de las declaraciones "condicionales" de JavaScript, que se utilizan para realizar diferentes acciones basadas en diferentes condiciones. Use el interruptor para seleccionar uno de los muchos bloques de código que se ejecutarán.

throw()

La throw declaración le permite crear un error personalizado. La throw instrucción arroja (genera) un error.

try...catch()

Cuando ocurre un error , JavaScript se detendrá y generará un mensaje de error.

var()

La var instrucción declara una variable. Las variables son contenedores para almacenar información.

while()

La while instrucción crea un bucle (alrededor de un bloque de código) que se ejecuta mientras se cumple una condición true.
