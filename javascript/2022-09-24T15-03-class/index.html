JavaScript Class

Publicado el 24 de septiembre de 2022

por Jezer Mejía

Las clases en JavaScript son similares que la mayoría de implementaciones del paradigma de la programación orientada a objetos que existe en otros lenguajes de programación. En JavaScript tenemos tanto las clases como los objetos, los cuales pueden tener sus respectivas propiedades y métodos, la herencia, el polimorfismo, etc.

Una clase en JavaScript tiene la siguiente forma:

class Clase {
  propiedad1 = "";
  propiedad2 = 80;

  constructor() { /* ... */ }
  metodo() { /* ... */ }
}
Como puede observarse, se definen unas cuantas propiedades, un constructor, y un método. Lo básico de una clase.

Instanciar clases

Como ya deberías de saber, instanciar una clase significa crear objetos a partir de la clase, objetos que contienen las propiedades y métodos definidos en la clase. Al igual que en otros lenguajes de programación, en JavaScript podemos instanciar cuántos objetos se desee con la palabra clave: new.

Así, se crea un objeto:

const objeto = new Clase();

Propiedades y métodos

Las clases pueden definir sus propiedades que son datos únicos de cada objeto, y definir métodos que representarían las funciones de tal objeto.

Propiedades

En JavaScript, debido a que es un lenguaje interpretado y de tipaje no estricto, las propiedades por lo general no se definen en el cuerpo de la clase, sino en el constructor con su valor inicial o computado según los parámetros del constructor.

Ejemplo:

class User {
  constructor(name) {
    this.name = name;
  }
}

En el anterior ejemplo se define una propiedad llamada name en el constructor.

Nota: Para acceder a las propiedades y métodos dentro de una clase se usa la palabra clave: this

Métodos

A diferencia de una función cualquiera que está fuera de una clase, los métodos no requieren de una palabra clave como function para definirse; un método se define sólo con el nombre del método, los paréntesis que contiene 0 o más parámetros, y el cuerpo del método.

Ejemplo:

class User {
  constructor(name) {
    this.name = name;
  }

  sayHola() {
    console.log(`Hola ${this.name}!!`);
  }
}

En el ejemplo anterior, el método sayHola imprimiría en pantalla un sencillo Hola tu-nombre!!.

Getter/Setter

Las clases, a su vez, pueden definir propiedades encapsuladas o computadas con los getter y setters. Son útiles para limitar la accesibilidad de una propiedad al ser capaz de hacerlas sólo de lectura, o bien tanto de lectura como escritura.

Un getter se define como un método sin parámetros precedido de la palabra clave: get; un getter siempre debe retornar un valor. Mientras, un setter se define como un método con un parámetro precedido de la palabra clave: set.

Ejemplo:

class User {
  constructor(name) {
    this.name = name;
  }
  get name() {
    return this._name;
  }
  set name(value) {
    if (value == "Juan") {
      console.log("El nombre Juan no está permitido");
      return;
    }
    this._name = value;
  }
}

const juan = new User("Juan"); // El nombre Juan no está permitido
console.log(juan.name); // undefined

En este ejemplo se crea un objeto de la clase User con el nombre de Juan. Dentro del constructor se llama al setter y éste valida si el nombre es "Juan"; ya que lo es, se imprime en pantalla El nombre Juan no está permitido y el nombre queda indefinido.
