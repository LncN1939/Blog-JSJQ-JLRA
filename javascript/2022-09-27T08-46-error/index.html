JavaScript Error

Publicado el 27 de septiembre de 2022

por Roire Villavicencio

El objeto Error proporciona información de error cuando se produce un error.

name

La namepropiedad establece o devuelve el nombre de un error.

message

La message propiedad establece o devuelve un mensaje de error.
