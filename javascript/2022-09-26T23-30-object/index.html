JavaScript Object

Publicado el 26 de septiembre de 2022

por Jezer Mejía

Un objeto representa uno de los tipos básicos de datos de JavaScript, el cual es usado para almacenar datos y entidades más complejas. No hay que confundir este Objeto con una instancia de una clase, ya que Object hace referencia a una "clase" primaria y no es para nada una instancia.

Un objeto es la unidad básica de JavaScript (dicen muchos por ahí). La mayoría de objetos existentes en JavaScript son instancias de Object. De esta manera, cualquier cambio que se haga a la clase Object por medio de su prototype se verá reflejado en todos los objetos de JavaScript.

Constructor

Al igual que otras clases en JavaScript, el Object puede ser instanciado con cualquier parámetro de entrada que devolverá un resultado correspondiente a la entrada.

En el siguiente ejemplo, se crea un valor "booleano" con Object: 

const b = new Object(true);

Prototype

La propiedad prototype es una propiedad global de una clase que contiene la lógica básica de una instancia. Así, modificar esta propiedad en una clase implica estar modificando los prototype de los objetos y las clases hijas.

Por lo general, JavaScript ya ofrece un prototype por defecto en la clase Object de la cual todos extienden. Esta implementación resulta útil para ofrecer métodos y propiedades que pueden usarse en cualquier contexto, como lo puede ser el método: toString(), el cual siempre estará disponible incluso si la clase no lo define.

Está también el método valueOf que retorna el valor primitivo del objeto; lo ideal sería reemplazarlo por una implementación propia. Podemos saber si un objeto es enumerable con propertyIsEnumerable. Saber si tiene alguna propiedad con hasOwnProperty, etc.

En el siguiente ejemplo se crea una clase llamada Area que posee sus 2 dimensiones y una modificación a su prototype para hacerlo útil en otros contextos. 

class Area {
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }
  get area() {
    return this.x * this.y;
  }
}

Area.prototype.valueOf = function () {
  return this.area;
};

const area1 = new Area(5, 5); // Área de 25
console.log(area1 + 5); // 25 + 5
