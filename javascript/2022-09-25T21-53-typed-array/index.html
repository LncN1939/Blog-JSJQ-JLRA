Typed Array

Publicado el 25 de septiembre de 2022

por Armando Meza

Los typed arrays en Javascript son buffers de datos binarios similares a los arrays

Objetos Typed Array

Object Data Type Range
Int8Array Entero con signo (byte) -128/127
Uint8Array Entero sin signo (octeto) 0/255
Uint8ClampedArray Entero sin signo (octet) 0/255
Int16Array Entero corto -32768/32767
Uint16Array Entero corto sin signo 0/65535
Int32Array Entero largo con signo -231/231-1
Uint32Array Entero largo sin signo 0/232
Float32Array Flotante con 7 cifras significativas 1.2x10-38/3.4x1038
Float64Array Flotante con 16 cifras significativas 5.0x10-324/1.8x10308
BigInt64Array Entero grande con signo -263/263-1
BigUint64Array Entero grande sin signo 0/264

Ejemplo

const cadenaEnteros = new Int8Array(5);       
          // [0, 0, 0, 0, 0]
