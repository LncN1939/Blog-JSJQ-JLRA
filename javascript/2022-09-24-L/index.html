JS Math

Publicado el 24 de septiembre del 2022

por Leo Corea

En este post se explicara todo lo relacionado a JS MATH el cual es un objeto que permite realizar tareas matemáticas.
Las matemáticas no son un constructor. Todas las propiedades/métodos de Math se pueden llamar usando Math como un objeto, sin crearlo

Ejemplo

            let x = Math.PI;
            let y = Math.sqrt(16);

Metodos y propiedades de JS MATH

Math.abs()

Este metodo devuelve el valor absoluto de un numero

Sintaxis

            Math.abs(x)

Parametros

Parametro Descripcion
x Requiere un numero

Valores de retorno

Tipo Descripcion
Numero El valor absoluto del numero.
Nan si el valor no es un número y 0 si el valor es null.

Ejemplo

            Math.abs(-12.45)
