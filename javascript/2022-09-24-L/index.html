JavaScript Math

Publicado el 24 de septiembre del 2022

por Leo Corea

En este post se explicará todo lo relacionado a JS Math el cual es un objeto que permite realizar tareas matemáticas.
Las matemáticas no son un constructor. Todas las propiedades/métodos de Math se pueden llamar usando Math como un objeto, sin crearlo

Ejemplo

            let x = Math.PI;
            let y = Math.sqrt(16);

Metodos y propiedades de JS Math

Math.abs()

Este metodo devuelve el valor absoluto de un numero

Sintaxis

            Math.abs(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número El valor absoluto del número.
Nan si el valor no es un número y 0 si el valor es null.

Ejemplo

            Math.abs(-12.45)

Math.acos()

Es un metodo de devuelve el arcoseno (en radianes) de un numero entre el 0 y PI, asi mismo espera recibir un rango de numeros -1 y 1

Sintaxis

            Math.acos(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número 0 a PI.
NaN si el parámetro está fuera del rango de -1 a 1 o no es numérico.

Ejemplo

            Math.acos(-1)
            Math.acos(0)
            Math.acos(1)

Math.acosh()

Devuelve el arcocoseno hiperbólico de un número, si el parámetro es menor que 1, el método devuelve NaN.

Sintaxis

          Math.acosh(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número El arcocoseno hiperbólico del parámetro, se mostrara NaN si el parámetro es menor que 1 o no es un valor numérico.

Ejemplo

            Math.acosh(1)
            Math.acosh(2)

Math.asin()

Es un metodo de devuelve el arseno (en radianes) de un numero entre el -2/PI y 2/PI, asi mismo espera recibir un rango de numeros entre -1 y 1

Sintaxis

            Math.asin(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número -PI/2 a PI/2. Se obtendra NaN si el parámetro no es numérico o está fuera del rango de -1 a 1.

Ejemplo

            Math.asin(1)
            Math.asin(0)
            Math.asin(-1)

Math.asinh()

El Math.asinh()método devuelve el arco-seno hiperbólico de un número.

Sintaxis

            Math.asinh(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número El arco-seno hiperbólico del parámetro. Se obtendra NaN si el parámetro no es numérico.

Ejemplo

            Math.asin(12)

Math.atan()

Es un método que devuelve la arcotangente de un número como un valor entre -PI/2 y PI/2 radianes.

Sintaxis

            Math.atan(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número -PI a PI/2. Se obtendra NaN si el parámetro no es numérico o esta vacio.

Ejemplo

            Math.asin(14)

Math.atan2()

Este método devuelve el arcotangente del cociente de sus argumentos, con un valor numérico entre PI y -PI radianes. El número devuelto representa el ángulo en sentido antihorario en radianes (no grados) entre el eje X positivo y el punto (x, y).

Sintaxis

            Math.atan2(y,x)

Parámetros

Parámetro Descripción
x Requiere la coordenada x
y Requiere la coordenada y

Valores de retorno

Tipo Descripción
Número PI a -PI. Se obtendra NaN si los valores están vacíos o no son numéricos.

Ejemplo

            Math.asin(12, 4)

Math.atanh()

Este método devuelve el arcotangente hiperbólico de un número, que espera un numero entre -1 y 1 donde regresara Infinite cuando sea 1 y -Infinite cuando sea -1

Sintaxis

            Math.atanh(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número El arcotangente hiperbólico del número, o NaN, o Infinito, o -Infinito. Se obtendra NaN si el número es mayor que 1 o menor que -1.

Math.cbrt()

Devuelve la método devuelve la raíz cúbica de un número.

Sintaxis

            Math.cbrt(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número La raiz cubica de un numero

Ejemplo

            Math.cbrt(13)

Math.ceil()

Este metodo redondea redondea un número redondeado hacia arriba con el entero más cercano.

Sintaxis

            Math.ceil(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número El entero más cercano al número redondeando hacia ARRIBA.

Ejemplo

            Math.ceil(0.70)

Math.clz32()

Este método devuelve el número de ceros iniciales en un número binario de 32 bits. clz32 es la abreviatura de CountLeadingZeroes32

Sintaxis

            Math.clz32(x)

Parámetros

Parámetro Descripción
x Requiere un número

Valores de retorno

Tipo Descripción
Número Número de ceros iniciales en la representación binaria de 32 bits de un número. Si el número es 0, devuelve 32 (todos los bits son 0)

Ejemplo

            Math.asin(7)

Math.cos()

Devuelve el coseno de un numero, con un resultado numerico de entre -1 y 1, el cual espero nuneros en radianes

Que son los Radianes

Radianes es la cantidad α de rotación b de un ángulo en un círculo:


Grados PI Radianes
0° 0 0
1° PI/180 0.0175
30° PI/6 0.52
45° PI/4 0.79
90° PI/2 1.57
180° PI 3.14
360° PI*2 6.28

Sintaxis

            Math.cos(x)

Parámetros

Parámetro Descripción
x Requiere un número representado en radianes

Valores de retorno

Tipo Descripción
Número -1 a 1 El coseno del número. Se obtendra NaN si el parámetro no es numérico.

Ejemplo

            Math.cos(3.14)
            Math.cos(Math.PI)

Para usar grados conviertelos en radianes con degree

            Math.cos(degree * Math.PI / 180);
