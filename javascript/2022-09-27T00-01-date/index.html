JavaScript Date

Publicado el 27 de septiembre de 2022

por Leo Corea

El objeto Date trabaja con fechas y horas. Los objetos de fecha se crean con new Date().

new Date()

constructor

La propiedad constructor devuelve la función que creó el prototipo de Fecha. Para las fechas de JavaScript, la constructorpropiedad devuelve: 

función Fecha() { [código nativo] }
            
const l = new Date();
let text = l.constructor;

getDate

La propiedad getDate método devuelve el día del mes (1 a 31) de una fecha.

const l = new Date();
let day = l.getDate();

Asi mismo se puede implementar esta funcion para obtener años, hora, minutos ect.

Nombre Descripcion
getDay() Retorna un dia de la semana (0-6)
getFullYear() Retorna el año
getHours() Retorna la hora (0-23)
getMiliseconds Retorna milisegundos (0-999)
getMinutes() Retorna minutos (0-59)
getMoth() retorna mes (0-11)
getSeconds Retorna segundos (0-59)

getTimezoneOffset()

getTimezoneOffset() devuelve la diferencia entre la hora UTC y la hora local. Tambien devuelve la diferencia en minutos.

Por ejemplo, si su zona horaria es GMT+2, se devolverá -120

const l = new Date();
let diff = l.getTimezoneOffset();

getUTCDate()

getUTCDate() devuelve el día del mes (1 a 31) de un objeto de fecha, segun UTC

Que es UTC?

UTC (Universal Time Coordinated) es la hora establecida por el estándar de hora mundial. La hora UTC es la misma que la hora GMT (hora del meridiano de Greenwich).Todos los métodos getUTC de JavaScript asumen que la fecha es la hora local.

const l = new Date();
let day = l.getUTCDate();

Asi mismo se puede implementar esta funcion para obtener años, hora, minutos ect.

Nombre Descripcion
getUTCDay() Retorna un dia de la semana (0-6), deacuerdo con UTC
getUTCFullYear() Retorna el año, deacuerdo con UTC
getUTCHours() Retorna la hora (0-23), deacuerdo con UTC
getUTCMiliseconds Retorna milisegundos (0-999), deacuerdo con UTC
getUTCMinutes() Retorna minutos (0-59), deacuerdo con UTC
getUTCMoth() retorna mes (0-11), deacuerdo con UTC
getUTCSeconds Retorna segundos (0-59), deacuerdo con UTC

now()

now() devuelve el número de milisegundos desde el 1 de enero de 1970. El cual es inalterable 

let ms = Date.now();

parse()

parse() analiza una cadena de fecha y devuelve la diferencia horaria desde el 1 de enero de 1970. Y lo devuelve en milisegundos. 

let ms = Date.parse("September 26, 2022");

setDate()

setDate() establece el día del mes de una fecha. 

const l = new Date();
l.setDate(0)

Asi mismo se puede implementar esta funcion para usarla con años, hora, minutos ect.

Nombre Descripcion
setFullYear() Establece el año de una fecha
setHours() Establece la hora de una fecha
setMiliseconds Establece los milisegundos de una fecha
setMinutes() Establece el minutos de una fecha
setMoth() Establece el mes de una fecha
setSeconds Establece los segundos de una fecha

setUTCDate()

El método setUTCDate() establece el día de un objeto Date, según UTC (Universal time coordinated).

const l = new Date();
l.setUTCDate(16);

Asi mismo se puede implementar esta funcion para usarla con años, hora, minutos ect.

Nombre Descripcion
setUTCFullYear() Establece el año de una fecha, segun UTC
setUTCHours() Establece la hora de una fecha, segun UTC
setUTCMiliseconds Establece los milisegundos de una fecha, segun UTC
setUTCMinutes() Establece el minutos de una fecha
setUTCMoth() Establece el mes de una fecha, segun UTC
setUTCSeconds Establece los segundos de una fecha, segun UTC

toDateString()

toDateString() devuelve la fecha (no la hora) de un objeto de fecha como una cadena. 

const l = new Date();
let text = l.toDateString();

toJSON()

toJSON() devuelve un objeto de fecha como una cadena, con formato de fecha JSON. Las fechas JSON tienen el mismo formato que el estándar ISO-8601: AAAA-MM-DDTHH:mm:ss.sssZ. 

const l = new Date();
let text = l.toJSON();
