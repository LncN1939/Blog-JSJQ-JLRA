JavaScript Number

Publicado el 27 de septiembre de 2022

por Roire Villavicencio

Todos los métodos numéricos devuelven un nuevo valor. No cambian la variable original.

Métodos de Number

construtor

La constructor propiedad devuelve la función que creó el prototipo de Número.

isFinite()

El Number.isFinite() método devuelve truesi un número es un número finito.

isInteger()

El Number.isInteger() método devuelve truesi un valor es un número entero del tipo de datos Número.

isNaN()

En JavaScript, NaN es la abreviatura de "Not-a-Number". En JavaScript, NaN es un número que no es un número legal.

isSafeInteger()

El Number.isSafeInteger() método devuelve trues un número es un entero seguro.

MAX_VALUE

Number.MAX_VALUE devuelve el mayor número posible en JavaScript.

MIN_VALUE

Number.MIN_VALUE devuelve el número más pequeño posible en JavaScript.

NEGATIVE_INFINITY

Number.NEGATIVE_INFINITY devuelve infinito negativo.

NaN

En JavaScript, NaN es un número que no es un número legal.

POSITIVE_INFINITY

POSITIVE_INFINITY devuelve infinito positivo.

prototype

prototype le permite agregar nuevas propiedades y métodos a los números.

toExponential()

El método toExponential() convierte un número en una notación exponencial.

toFixed()

El toFixed() método convierte un número en una cadena.

toLocaleString()

Devuelve un número toLocaleString() como una cadena, utilizando el formato del idioma local.

toPrecision()

El toPrecision() método formatea un número a una longitud especificada.

toString()

JavaScript usa el toString() método internamente cuando un objeto debe mostrarse como texto (como en HTML), o cuando un objeto debe usarse como una cadena.

valueOf()

El valueOf() método devuelve el valor primitivo de un número.
